📊 Reference Checklist Summary

EN ISO 13849-2 annex checklists – check items below and the status updates automatically.

Annex A – Mechanical components

⚙ Shafts / Spindles

▶

Faults that can be excluded

  • Fracture due to fatigue
  • Fracture due to overload

Typical reference

  • EN 13849-2 Table A.4
  • Safety factor ≥ 2 against fracture limit

Checklist for fault exclusion

None checked yet

📋 Documentation for technical file

  • Strength calculation (FEM or analytical)
  • Material certificate EN 10204 type 3.1
  • Drawing with radius specifications
  • Fatigue analysis / service life calculation

⚙ Gears / Gearboxes

▶

Faults that can be excluded

  • Tooth fracture
  • Separation of gear from shaft

Typical reference

  • EN 13849-2 Table A.4
  • Service factor fB ≥ 2.0

Checklist for fault exclusion

None checked yet

📋 Documentation for technical file

  • Manufacturer datasheet (e.g. SEW, Nord)
  • Calculated service factor
  • Lubrication schedule

⚙ Chains / Belts

▶

Faults that can be excluded

  • Fracture (rarely permitted as sole safeguard)

Typical reference

  • EN 13849-2 Table A.4
  • Safety factor ≥ 10 (chain), ≥ 8 (belt)

Checklist

None checked yet

📋 Documentation

  • Load calculation
  • Maintenance log / interval plan
  • Manufacturer data with breaking load

⚙ Fasteners (Screws / Bolts)

▶

Faults that can be excluded

  • Loosening due to vibration
  • Fatigue fracture

Typical reference

  • EN 13849-2 Table A.4
  • VDI 2230 (bolted joint calculation)

Checklist

None checked yet

📋 Documentation

  • Assembly instruction with torque values
  • Torque record (verification)
  • VDI 2230 calculation for critical joints

⚙ Springs

▶

Faults that can be excluded

  • Fracture (fatigue)
  • Permanent deformation (set)

Typical reference

  • EN 13849-2 Table A.4
  • EN 13906-1 (compression springs)

Checklist

None checked yet

📋 Documentation

  • Spring calculation (force, deflection, life)
  • Manufacturer data / datasheet
  • Material certificate

⚙ Cam Mechanisms / Cam Discs

▶

Faults that can be excluded

  • Fracture
  • Wear leading to functional failure

Typical reference

  • EN 13849-2 Table A.4
  • Surface treatment and hardening

Checklist

None checked yet

📋 Documentation

  • Load calculation
  • Hardening protocol
  • Lubrication schedule

⚙ Guides / Linear Units

▶

Faults that can be excluded

  • Jamming / blocking
  • Fracture of mounting

Typical reference

  • EN 13849-2 Table A.4

Checklist

None checked yet

📋 Documentation

  • Manufacturer data with load capacity
  • Maintenance intervals
Annex B – Pneumatic components

💨 Pneumatic Cylinders

▶

Faults that can be excluded

  • Cylinder body fracture
  • Piston rod fracture

Faults that CANNOT normally be excluded

  • Internal leakage (seal wear)
  • External leakage

Checklist for exclusion of body fracture

None checked yet

📋 Documentation

  • Datasheet with pressure rating
  • Declaration of conformity with ISO 15552/6432
  • Operating parameters (pressure, temperature)

💨 Valves (directional / control valves)

▶

Faults that can be excluded

  • Body fracture
  • Manual override stuck (if correctly dimensioned)

Faults that CANNOT normally be excluded

  • Failure to switch (solenoid / spring failure)
  • Internal leakage (spool wear)

Checklist for exclusion of body fracture

None checked yet

📋 Documentation

  • Datasheet with pressure and temperature range
  • Pneumatic circuit diagram
  • Filter specification

💨 Pipes / Hoses

▶

Faults that can be excluded

  • Fracture of metal pipe (if correctly dimensioned)

Faults that CANNOT normally be excluded

  • Hose burst (ageing)
  • Leakage at fittings

Checklist

None checked yet

📋 Documentation

  • Datasheet with burst pressure
  • Hose replacement intervals
  • Pneumatic circuit diagram

💨 Pressure Regulators / Pressure Limiters

▶

Faults that can be excluded

  • Body fracture

Faults that CANNOT normally be excluded

  • Full open (pressure breakthrough)
  • Internal leakage

Checklist

None checked yet

📋 Documentation

  • Datasheet with pressure range
  • Pneumatic diagram with safety/relief valve

Annex D – Electrical / Electronic components

⚡ Cables / Conductors

▶

Faults that can be excluded

  • Short circuit between conductors
  • Short circuit to earth/PE

Faults that CANNOT normally be excluded

  • Open circuit (cable break)

Checklist

None checked yet

📋 Documentation

  • Cable specification (cross-section, insulation, temperature class)
  • Cable routing drawing
  • Insulation test record

⚡ Reläer / Kontaktorer

⚡ Sensors / Transducers / Limit Switches

▶

Faults that can be excluded

  • Short circuit output – supply (with special design)
  • Body fracture (with correct IP class)

Faults that CANNOT normally be excluded

  • No actuation (missed object)
  • Permanently ON or OFF

Checklist

None checked yet

📋 Documentation

  • Datasheet with IP class and contact data
  • Mounting drawing
  • Circuit diagram

⚡ Circuit Boards / PCB

▶

Faults that can be excluded

  • Solder joint fracture (with correct process)

Faults that CANNOT normally be excluded

  • Component failure (IC, transistor)
  • Short circuit between tracks

Checklist

None checked yet

📋 Documentation

  • Soldering standard / IPC reference
  • EMC test report
  • Layout safety analysis

⚡ Connectors / Plugs

▶

Faults that can be excluded

Faults that CANNOT normally be excluded

Checklist

None checked yet
